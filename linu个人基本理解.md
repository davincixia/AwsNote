# linu个人基本理解

## **学命令之前的理解**



**命令** 和 **你下载好的可执行文件**（服务或者应用）一样 （比如alist） 都是某些编程语言通过编译产生的 二进制可执行文件，

不同的是 bin sbin 里的一些可执行文件 （也就是我们用的命令 ls pwd） 也是如此 只是编程很底层 c语言写的。

.sh文件 （shell）脚本文件就是 系统命令的组合  而我们的程序就是主流语言编译生成的二进制可执行文件



**总结**

三类  **系统二进制 （命令）** **shell脚本文件（系统命令组合）**  **通常可执行文件**（主流语言生成的可执行二进制文件）



用法上的不同

**系统命令** ： 直接裸用 任何目录下 直接pwd或者ls

**你安装的服务**：比如   ./alist    ./ 表示你是在当前目录下 所以你需要切换到该服务（应用）的相关目录下

**脚本文件**（.sh）: 

chmod+x xxx.sh  先使其可以被执行 赋予权限

然后就是同安装服务一样  ./xxx.sh

## **终端 shell 和不同shell**

我们写命令 肯定是在终端。 shell 充当 内核  我们编程语言  和 内核之间翻译官的位置。

shell 也有不同版本 shell不是跟随系统概念 任何系统都可以有shell 看发挥的作用。

win下 cmd powershell
